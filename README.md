# AutoHotkey US QWERTY French Accents

Use Alt-based shortcuts to type French accented characters on a US QWERTY layout (no dead keys).

This script keeps the standard US keyboard layout intact and adds optional Alt combinations for accented characters.

---

## Requirements

* Windows
* [AutoHotkey](https://www.autohotkey.com/) installed

---

## Installation

1. Install AutoHotkey from the official website.
2. Download or clone this repository.
3. Copy the `.ahk` file into your Windows Startup folder:

Quick way to open the Startup folder:

* Press `Win + R`
* Type:

```
shell:startup
```

* Press Enter

4. Double-click the `.ahk` file to launch the script.

---

## Automatic Startup

Placing the script in the Startup folder ensures it launches automatically each time Windows starts.

---

## Usage

Hold **Alt** and press the corresponding key to insert accented characters.

Examples (may vary depending on mapping):

* `Alt + e` → é
* `Alt + r` → è
* `Alt + a` → à
* `Alt + c` → ç
* `Alt + Shift e` → É
* and more for ù, ô, ï and shift version

No dead keys are used.

---

## Stopping the Script

* Right-click the green AutoHotkey icon in the system tray
* Click **Exit**

---

## Customization

You can open the `.ahk` file in any text editor and adjust the key mappings to fit your preferences.
